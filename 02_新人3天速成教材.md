# 【第二部分】新人 3 天速成版（快速上線用）

本手冊設計給「完全不懂冷氣」的新人，目標是 3 天後能在門市「看起來像專業的」，且不會講錯話被告。

---

## 📅 Day 1｜基礎觀念：不懂一定會亂賣

### 1. 分離式冷氣怎麼運作（白話版）

想像冷氣是一個「搬運工」：

* **室內機（上面的）**：把房間裡的「熱」吸進去。
* **冷媒（管子跑的）**：把吸到的熱搬出去。
* **室外機（外面的）**：把搬出來的熱排掉（所以室外機要通風，不然熱排不掉，裡面就不會涼）。
* **重點**：冷氣不是「製造冷氣」，而是「移走熱氣」。所以房間窗戶沒關好，熱氣一直跑進來，搬運工搬不完，就不會涼。

### 2. 變頻是什麼 / 不是什麼

* **是什麼**：像開車踩油門，熱的時候全速跑（快冷），涼了就輕踩油門維持速度（省電、恆溫）。
* **不是什麼**：不是「一開機就省電」。如果你房間西曬頂樓熱得要死，買太小台，它會一直全速跑，反而更耗電。
* **定頻是什麼**：像開車只有「全速衝」跟「停車」兩種模式。太冷就停，太熱就衝，所以忽冷忽熱又耗電。

### 3. 能源標示怎麼解釋才安全

* **CSPF**：數值「越高越好」（越省電）。
* **每年耗電量**：標示上的數字是「假設一年吹 1200 小時」算出來的。
* **話術**：「大哥，這台是能源效率第一級，CSPF 越高越省電，像這台已經超越國家標準了。」
* **⚠️ 禁語**：「你看這個數字，一年電費只要 xxx 元。」（絕對禁止！因為每個人使用習慣不同，電價也不同）。

### 4. 新人「不能自己回答」的問題清單（遇到請轉老鳥）

1. **「我家 5 坪要裝多大？」**（除非你能確定無西曬、無頂樓、無挑高，否則請客人給照片或預約場勘）。
2. **「安裝要多少錢？」**（只能給「基本安裝」表，額外管線、洗洞都要師傅現場報價）。
3. **「這台保證安靜嗎？」**（每個人對聲音敏感度不同，只能說「這是靜音機型」）。
4. **「能不能把室外機裝在廁所/陽台內？」**（散熱不良會不冷，且原廠可能不保固）。

### 📝 Day 1 新人必背重點
>
> * 冷氣是「移熱」不是「製冷」。
> * 變頻是「維持恆溫才省電」。
> * CSPF 越高越好。
> * 坪數、安裝費、非標準安裝位置 → 絕對不要自己拍胸脯保證。

## 📅 Day 2｜品牌差異速懂：客人問這台那台有什麼差

### 1. 六大品牌一句話定位

* **大金**：空調界的愛馬仕，最舒適（溫濕雙控），服務最好，價格最高。
* **日立**：台灣長青樹，最乾淨（凍結洗淨），最多人買，品質穩。
* **國際**：省電資優生，最科技（nanoe + IoT），功能給最多，CP 值高。
* **富士通**：懶人救星，唯一濾網自動掃，全機 10 年保固，耐操。
* **LG**：顏值變頻專家，最漂亮（鏡面），有 WiFi + UV 殺菌，年輕人愛。
* **禾聯**：抗沼氣之王，最便宜（高 CP 值），房東與惡劣環境首選。

### 2. 不同客人 → 適合哪個品牌（配對題）

* **家有嬰兒/怕冷風直吹** 👉 **大金**（康達氣流 + 溫濕雙控）。
* **潔癖/怕冷氣有霉味** 👉 **日立**（凍結洗淨）、**富士通**（自動掃濾網）。
* **精打細算/看數據省電** 👉 **國際**（CSPF 通常最高）。
* **裝潢漂亮/要黑色冷氣** 👉 **LG**（ArtCool 系列）。
* **我要最便宜/裝出租房** 👉 **禾聯**。
* **住北投/水溝邊** 👉 **禾聯**（沼氣防護）。

### 3. 常見錯誤迷思（論壇 vs 官方）

* **迷思**：「大金是泰國做的比較爛？」
  * **正解**：大金泰國廠是全球最新銳工廠，品質跟日本一樣，不用擔心。
* **迷思**：「買冷氣要買大一級才夠冷？」
  * **正解**：基本正確，特別是現在極端氣候，"剛好" 通常都會變 "不夠"，寧大勿小（變頻會自己降轉，可以省電，小台硬拖反而耗電）。
* **迷思**：「自動洗淨功能就不用再洗冷氣？」
  * **正解**：錯！那是延緩變髒的時間，不用每年洗，但 3-5 年還是要大保養一次。

### 📝 Day 2 新人必背重點
>
> * 怕吵怕風 👉 大金。
> * 怕髒怕洗 👉 日立、富士通。
> * 省電科技 👉 國際。
> * 外型 WiFi 👉 LG。
> * 便宜耐操 👉 禾聯。
> * 寧且選大一級，不要剛剛好。

## 📅 Day 3｜門市實戰：降低客訴與安全下莊

### 1. 標準銷售流程（SOP）

1. **問坪數與環境**：「請問要裝在客廳還是房間？大約幾坪？有沒有西曬或頂樓？」（這步錯，後面全錯）。
2. **問需求**：「特別在意省電、安靜、還是空氣清淨功能？」
3. **推品牌**：根據 Day 2 的配對推薦 1-2 個品牌（不要超過 3 個，客人會選擇障礙）。
4. **談價格與安裝**：說明機子價格 + 「基本安裝」範圍，強調「超出部分實作實收」。
5. **成交與預約**。

### 2. 新人最常講錯的 10 句話（❌絕對禁止）

1. 「這台絕對安靜，沒聲音。」（人耳總聽得到，改說：這台有靜音模式，非常小聲）。
2. 「電費一個月只要 xxx 塊。」（改說：這是一級能效，比舊型省電很多）。
3. 「安裝費全包了。」（絕對有額外費用，改說：含基本安裝 5 米銅管）。
4. 「今天訂明天裝。」（要看師傅檔期，改說：幫您查詢最快安裝時間）。
5. 「這台保固內什麼都免費。」（改說：主要零件原廠保固，但天災、人為、遙控器耗材除外）。
6. 「可以用 20 年。」（改說：正常保養使用 10 年以上沒問題）。
7. 「噸數抓剛好就好。」（改說：建議稍微大一點點，冷得快又省電）。
8. 「大金是日本做的。」（其實多是泰國，誠實說泰國製也是高品質）。
9. 「這台可以除濕不用買除濕機。」（改說：它有除濕效果但會降溫，冬天還是要除濕機）。
10. 「所有功能我都會教你。」（改說：基本操作我教您，太深奧的功能說明書都有）。

### 3. 不知道時的安全回應方式

* 「這個技術比較專業，我馬上幫您查一下原廠手冊確認。」（拿出手機/平板）。
* 「關於安裝的詳細費用，我們的師傅到現場會先報價，您同意才施工，不用擔心。」

### 4. 降低客訴的關鍵行為

* **醜話說在先**：安裝費實作實收、特殊環境（西曬）噸數要加大、旺季安裝要等。
* **不要過度承諾**：不要為了成交答應你做不到的事（例如：保證明天裝、保證絕不滴水）。
* **確認現場**：如果不確定坪數，強烈建議客人「拍照」或「預約看現場」。

### 📝 Day 3 新人必背重點
>
> * 先問環境（坪數/西曬）再推機器。
> * 安裝費永遠是「基本安裝 + 實作實收」。
> * 不講「絕對」、「保證」、「一定」。
> * 不確定就查，不要亂掰。
